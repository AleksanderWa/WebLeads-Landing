# WebLeads Hero Demo Video — Storyboard

**Purpose:** Landing-page hero demo + Notion blueprint asset.
**Format:** Animated UI mockup, ~34s, 1920×1080, silent (autoplay-friendly), CTA end card.
**Built with:** HeyGen HyperFrames. Project ID: `a20d57a3-a767-4e0a-82f7-920e349a8996`
**Vibe:** Slick product tour — purposeful motion, real UI labels, satisfying email-reveal peak.

## Demo scenario
ICP: small marketing agency / freelancer. Vertical shown: **HVAC contractors**.
Flow: one search → smart website filter ("emergency service") → AI-qualified results → owner's verified email → export.

## Brand
- Burgundy `#8A3033` (buttons, headings, logo)
- Gold `#D8B26E` (toggles, badges, highlights)
- Dark `#2D3142` (text)
- Warm light bg `#F8F6F4`, green `#2E7D5B` (verified states)
- Inter, 12px radius, soft shadows, whitespace.
